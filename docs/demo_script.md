# AutoNetIR Demo Script

1. Open the Streamlit dashboard.
2. Upload a PCAP sample.
3. Click **Start Analysis**.
4. Show the summary metrics and protocol chart.
5. Open one detected attack card.
6. Explain the method badge:
   - Signature means a known rule matched.
   - Behavior means the activity was abnormal compared with peers.
   - Hybrid means both matched.
7. Show signature evidence, behavior evidence, recommendation, and limitation.
8. Export JSON or CSV alerts.
